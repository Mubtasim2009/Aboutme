## Setup and Deployment

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Mubtasim2009/Aboutme.git
   cd Aboutme
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local web server (recommended):
     ```bash
     python -m http.server 8000
     # or
     npx serve
     ```

3. Navigate to `http://localhost:8000` in your browser

### Deployment

This website is hosted on **Cloudflare Pages** and can be deployed to any static hosting service:

- **Cloudflare Pages** (Current hosting)
- GitHub Pages
- Netlify
- Vercel
- Any static web hosting service

Simply upload the files to your hosting provider or connect your repository for automatic deployments.

## Customization

To customize this portfolio for your own use:

1. **Personal Information**: Edit the content in `index.html`
   - Update name, bio, and skills
   - Modify social media links
   - Add or remove certifications

2. **Styling**: Modify the `<style>` section in `index.html`
   - Change colors, fonts, and spacing
   - Adjust animations and transitions

3. **Background Images**: Replace images in `assets/background/`
   - Use WebP format for best performance
   - Maintain aspect ratio for seamless transitions

4. **Branding**: Update logo and icon in `assets/brand/`
